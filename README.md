# サーブレット & JSP 入門演習

## 演習1：`@WebServlet("/Yattemiru")`(servletTry.java)

### 🎯 目的
サーブレットの基本機能を習得し、**ユーザーのリクエストに応じてサーバー側のデータを動的に表示する**Webアプリケーションを構築します。

---

### 💻 アプリの概要
サーバー上に保存されたCSVデータを読み込み、ユーザーが指定した日付に基づいて情報を検索・表示します。

#### 🛠️ 処理の流れ
| フェーズ | 役割 | 内容 |
| :--- | :--- | :--- |
| **準備** | データ管理 | サーバー側にCSV形式のデータファイルを配置 |
| **初期化 (`init`)** | データロード | アプリ起動時にCSVを読み込み、データクラスに格納 |
| **表示 (`doGet`)** | 入力画面 | 日付入力フォームと検索ボタンをブラウザに表示 |
| **実行 (`doPost`)** | 検索・出力 | 指定された日付をキーにデータを抽出、結果をWebページに反映 |

## 演習2：`@WebServlet("/population")`(PopulationServlet.java)

### 🎯 目的
1の応用で、**ユーザーのリクエストに応じて外部サーバーからデータを取得、加工して表示する** Webアプリケーションを構築します。<br>
フォームへのテキスト入力に対して、クロスサイトスクリプティング対策も実装しました。

### 💻 アプリの概要
フォームで入力された市町村名とサーバー上に保存されたCSVデータからコードを特定し、e-StatのAPIを使って該当市町村の人口を取得
Webページに表示表示します。

#### 🛠️ 処理の流れ
| フェーズ | 役割 | 内容 |
| :--- | :--- | :--- |
| **準備** | 市町村コード取得 | e-Statサイトからデータを取得(取得方法はAIに相談)<br>市町村名と都市コードを抜き出してCSV形式ファイルにしてサーバー側に配置 |
| **初期化 (`init`)** | データロード | アプリ起動時にCSVを読み込み、データクラスに格納 |
| **表示 (`doGet`)** | 入力画面 | 日付入力フォームと検索ボタンをブラウザに表示 |
| **実行 (`doPost`)** | 検索・出力 | 指定された日付をキーにデータを抽出、結果をWebページに反映 |
